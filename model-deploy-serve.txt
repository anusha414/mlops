Model Deployment:-
Deploying the model that is generated by the data scientists n their local machine to a production server so that it is accessible to the end users.
From an MLOps (Machine Learning Operations) perspective, model deployment is the process of taking a trained machine learning model and making it available 
for use in a production environment where it can generate predictions for real users, applications, or business processes.

Model serving:-
Just deploying the model in the production deos not mean it is serving end users, you should take crae of model serving too.
1. Setting up the model with the required runtime for it and providing it with neccessary resources.
2. Setting up API for end users, load balancers, ingress controllers, API gateway and CDN.
